# Deleting client side mods
WARNING: you must run this in the mods folder

command for Unix/Linux based systems
```
rm -f \
CraftPresence-*.jar \
moreoverlays-*.jar \
ReAuth-*.jar \
rubidium-*.jar \
rubidium_extras-*.jar \
dynamiclightsreforged-*.jar
```
command for michealshaft binbows command prompt (NOT TESTED!)
```
del /F /Q /S ^
CraftPresence-*.jar ^
moreoverlays-*.jar ^
ReAuth-*.jar ^
rubidium-*.jar ^
rubidium_extras-*.jar ^
dynamiclightsreforged-*.jar
```
command for michealshaft powershell (NOT TESTED!)
```
rm -fo \
CraftPresence-*.jar \
moreoverlays-*.jar \
ReAuth-*.jar \
rubidium-*.jar \
rubidium_extras-*.jar \
dynamiclightsreforged-*.jar
```
